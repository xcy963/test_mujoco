# 记录一些新的语法

## 1cpp的线程唤醒工具
```cpp
plot_cv_.wait_for(lock, std::chrono::milliseconds(100), [this]() {//10HZ
    return !plot_queue_.empty() || !plot_running_;//使用谓词逻辑,当条件满足的时候就会调用
});

```
> 这个方法是线程主动休眠等待(条件满足的时候立刻唤醒),当然我们可以在其他地方写,让他被动唤醒

```cpp
plot_cv_.notify_one();//因为我们只有一个线程,所以这个使用一个的
```
> 在别的地方唤醒他,让他不再等待
